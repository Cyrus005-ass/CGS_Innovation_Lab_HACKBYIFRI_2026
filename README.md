 Description du projet

StuLink Bénin est une application web mobile-first conçue pour aider les étudiants béninois à :

trouver une colocation sécurisée, filtrée et adaptée

obtenir des stages académiques validés et centralisés

Elle combine deux modules essentiels :

 Module Colocation

 Module Stages

 Équipe du projet – CGS Innovation Lab
 Problématiques ciblées
Colocation

Loyers élevés à proximité des universités

Difficulté à trouver un colocataire compatible

Risques liés à colocation mixte

Arnaques fréquentes sur les réseaux sociaux

Absence de filtrage sécurisé

Stages académiques

Absence de plateforme centralisée

Favoritisme dans l’attribution des stages

Manque de transparence

Difficulté pour les entreprises à toucher les bons profils

 Solution – StuLink Bénin
 Module Colocation

Publication d’annonces structurées

Recherche avancée avec filtres (budget, quartier, genre, distance)

Mise en relation directe via WhatsApp

Filtrage et sécurité (profil étudiant validé)

 Module Stages

Consultation d’offres validées

Matching simple basé sur la filière

Publication d’offres par entreprises et établissements

Validation administrative avant publication
 Fonctionnalités clés
 Inscription & Authentification

Données requises : nom, prénom, école, filière, année, NPI, photo carte étudiante (optionnelle)

Compte en attente de validation

Acceptation obligatoire des CGU

 Module Colocation

Publication d’annonces filtrées

Recherche intelligente

Filtrage par genre et option colocation mixte
 Module Stages

Demande validée administrativement
